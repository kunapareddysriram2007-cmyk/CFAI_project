# CFAI-Project

Small Python project containing `main1.py`.

## Description

This repository contains a simple script `main1.py`. The README provides quick setup and run instructions.

## Requirements

Install dependencies from `requirements.txt`:

```bash
python -m venv .venv
# Windows
.venv\\Scripts\\activate
# macOS / Linux
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

Run the main script:

```bash
python main1.py
```
